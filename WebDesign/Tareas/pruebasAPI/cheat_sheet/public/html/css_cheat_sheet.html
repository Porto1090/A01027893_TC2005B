CSS Cheat Sheet

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico que se utiliza para dar estilo a los elementos de una página web.

Permite definir la presentación de los elementos de una página web, como el diseño, el formato y el estilo de los elementos HTML.

Border

Propiedad que permite definir el borde de un elemento.
Border: "Solid"
Border: "Dashed"
Border: "Dotted"
Border: "Double"
Border: "Ridge"
Border: "Groove"

Border-left, border-right, border-top, border-bottom

Propiedad que permite definir el borde izquierdo de un elemento, puede usar los estilos de borde.
Propiedad que permite definir el borde derecho de un elemento, puede usar los estilos de borde.
Propiedad que permite definir el borde superior de un elemento, puede usar los estilos de borde.
Propiedad que permite definir el borde inferior de un elemento, puede usar los estilos de borde.

Border-Radius

Propiedad que permite definir el radio de las esquinas de un elemento.

Background Image

Propiedad que permite definir una imagen de fondo para un elemento.

Color

Propiedad que permite definir el color del texto de un elemento. Puede ser en diferentes formatos como RGB, HEX, HSL.

Font-Size

Propiedad que permite definir el tamaño de la fuente de un elemento.

Font-Family

Propiedad que permite definir el tipo de fuente de un elemento.

Font-Style

Propiedad que permite definir el estilo de la fuente de un elemento.

Font-Weight

Propiedad que permite definir el grosor de la fuente de un elemento.

Width

Propiedad que permite definir el ancho de un elemento, bloque.

Height

Propiedad que permite definir la altura de un elemento, bloque.

List-style-type

Propiedad que permite definir el estilo de la lista de un elemento.

Margin

Propiedad que permite definir el margen de un elemento.

Padding

Propiedad que permite definir el relleno de un elemento.

Overflow

Propiedad que permite definir el comportamiento de un elemento cuando su contenido es más grande que su tamaño.

Display

Propiedad que permite definir el tipo de visualización de un elemento.

Position

Propiedad que permite definir la posición de un elemento.

Filter

Propiedad que permite aplicar efectos visuales a un elemento.
