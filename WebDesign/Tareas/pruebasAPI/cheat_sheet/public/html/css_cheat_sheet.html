CSS Cheat Sheet

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico que se utiliza para dar estilo a los elementos de una página web.

Permite definir la presentación de los elementos de una página web, como el diseño, el formato y el estilo de los elementos HTML.

Border

Propiedad que permite definir el borde de un elemento.

Border-left, border-right, border-top, border-bottom

Propiedad que permite definir el borde izquierdo de un elemento.
Propiedad que permite definir el borde derecho de un elemento.
Propiedad que permite definir el borde superior de un elemento.
Propiedad que permite definir el borde inferior de un elemento.

Border-Radius

Propiedad que permite definir el radio de las esquinas de un elemento.

Background Image

Propiedad que permite definir una imagen de fondo para un elemento.
